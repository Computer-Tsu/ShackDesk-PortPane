# Code of Conduct

## Our Pledge

We are committed to making participation in PortPane a welcoming experience for everyone, regardless of age, experience level, or background.

## Standards

**Be kind.** Critique code, not people. Assume good intent.

**Be patient.** Many contributors and users are non-technical ham radio operators. Adapt your communication accordingly.

**Be helpful.** If you see a question you can answer, answer it.

## Enforcement

Violations of this code of conduct may be reported to the project maintainer via [GitHub Discussions](https://github.com/Computer-Tsu/shackdesk-portpane/discussions) or directly via GitHub profile contact.

Consequences range from a warning to a permanent ban from the project, at the maintainer's discretion.

This policy applies in all project spaces: issues, pull requests, discussions, and any other venue used by this project.
